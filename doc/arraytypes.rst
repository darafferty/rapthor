Array Types
===========


The Array classes are dynamic multi-dimensional arrays with a fixed number of dimensions.
They can be owning, freeing the memory upon destruction, or non-owning, leaving
the resposibility for deallocation to the user.
Arrays are non-copyable, but they can be moved.
